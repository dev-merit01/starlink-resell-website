# starlink-resell-website
Responsive and fully functional website for a local company called KuiperTech that specializes in reselling insatlling Starlink kits to the Zimbabwean community

## OVERVIEW
This Website Showcase:
- Starlink kit resell and installation
- Wifi setup and configuration
- After-sales support and maintenance
- Contact and inquiry form

## BUILT WITH
- **HTML5**
- **CSS3** Responsive Design
- **Javascipt** (Interactive features)
